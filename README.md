# AI学习项目简介

本项目包含人工智能导论课程相关作业与深度学习 PyTorch 入门代码，适合初学者快速上手 AI 基础知识与实践。

## 目录结构

- `hw2/`：C++ 文件读写等基础编程作业
- `hw3/`：基于 OpenAI/DeepSeek API 的文本摘要应用与测试
- `hw4/`：C++ 相关练习
- `py.learning/`：Python 基础学习代码
- `pytorch/`：PyTorch 深度学习入门示例（如梯度下降、线性回归等）
